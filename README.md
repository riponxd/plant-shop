## Getting Started

1. **Clone the repository:**
   ```powershell
   git clone https://github.com/ShahriarTbc/plant-shop.git
   cd plant-shop
   ```

2. **Install dependencies:**
   ```powershell
   npm install
   ```

3. **Start the development server:**
   ```powershell
   npm start
   ```

4. **Build for production:**
   ```powershell
   npm run build
   ```
